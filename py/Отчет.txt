Проект: Разработка системы распознавания автомобильных номеров на Python


Система для автоматического обнаружения и распознавания российских автомобильных номеров на изображениях. Включает этапы предварительной обработки изображений и постобработки распознанного текста для повышения точности.

1. Инициализация моделей
YOLOv8 (yolov8n.pt) для обнаружения номерных знаков

EasyOCR с поддержкой английского и русского языков для распознавания текста

2. Функционал системы
Предобработка изображений:

CLAHE (адаптивное выравнивание гистограммы)

Адаптивная пороговая обработка

Морфологические операции

Проверка формата номеров:

Регулярные выражения для валидации российских номеров

Коррекция распознанного текста под стандартный формат

Постобработка текста:

Удаление лишних символов

Приведение к верхнему регистру

Форматирование по шаблону

3. Обработка изображений
Поддерживаются форматы: JPG, JPEG, PNG

Автоматический поиск в папке 'images'

Детальная консольная информация о каждом обработанном изображении

4. Улучшенное распознавание текста
Каскадная обработка изображения перед распознаванием:

Градации серого

Билатеральный фильтр

CLAHE

Адаптивный порог

Морфологическое закрытие


Логика работы
Поиск всех изображений в указанной папке

Детекция номерных знаков с YOLOv8

Интенсивная предобработка области номера

Распознавание текста

Постобработка и валидация распознанного текста

Визуализация результатов

Сохранение метаданных

Экспорт всех результатов в JSON-файл


Выходные данные
Изображения с bounding boxes и распознанным текстом (в папке output)

Файл results.json с полными результатами обработки

Подробный вывод в консоль о каждом обработанном номере
